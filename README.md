# 🔐 Gerenciador de Senhas em Python

Projeto desenvolvido durante meus estudos em **Análise e Desenvolvimento de Sistemas** com foco em lógica de programação, validação de dados e boas práticas de segurança.

O sistema analisa a força de uma senha, verifica se ela atende aos principais critérios de segurança e apresenta sugestões para torná-la mais forte.

---

# 🚀 Funcionalidades

- Verificação do tamanho mínimo da senha
- Identificação de letras maiúsculas
- Identificação de letras minúsculas
- Verificação de números
- Verificação de caracteres especiais
- Sugestões de melhoria para senhas fracas
- Validação completa da força da senha

---

# 🛠 Tecnologias utilizadas

- Python 3
- Estruturas condicionais (`if`)
- Estruturas de repetição (`for`)
- Métodos de String
- Operadores Lógicos
- Booleans (`True` e `False`)

---

# ▶️ Como executar

Clone o projeto

```bash
git clone https://github.com/LuizAveiro/Gerenciador-de-Senhas.git
```

Entre na pasta

```bash
cd Gerenciador-de-Senhas
```

Execute

```bash
python gerenciador_senhas.py
```

# 🔒 Regras utilizadas

Uma senha é considerada forte quando possui:

- Pelo menos 12 caracteres
- Pelo menos uma letra maiúscula
- Pelo menos uma letra minúscula
- Pelo menos um número
- Pelo menos um caractere especial
